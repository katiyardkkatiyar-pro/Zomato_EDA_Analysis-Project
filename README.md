# 🍽️ Zomato Bangalore Restaurants - Exploratory Data Analysis

## 📌 Project Overview
This project performs a comprehensive Exploratory Data Analysis (EDA) on Zomato's Bangalore restaurant dataset containing **100,000+ restaurants** to uncover actionable insights about the Bangalore food scene.

## 🎯 Key Objectives
- Analyze restaurant ratings, pricing, and popularity trends
- Identify factors affecting restaurant ratings
- Find best value-for-money restaurants in Bangalore
- Understand impact of online delivery and table booking on ratings

## 📊 Dataset Information
- **Total Records:** 100,000
- **Features:** 17 columns (restaurant info, ratings, cost, services)
- **Source:** Zomato Bangalore

## 🔍 Key Insights
| Insight | Finding |
|---------|---------|
| Average Rating | 3.70/5.0 |
| Average Cost for Two | ₹1,097 |
| Most Common Rating | Good (3.0-3.9) - 58.2% |
| Online Delivery Impact | +0.15 rating points higher |
| Table Booking Impact | +0.22 rating points higher |
| Zomato Pro Impact | +0.08 rating points higher |
| Top City | Bangalore (all data) |
